## 🔧 接口功能详情

### 1. fund-detail（基金详情）

**功能**: 获取基金完整信息

**返回数据**:
- ✅ 基本信息（代码、名称、类型、风险等级）
- ✅ 基金公司、基金经理
- ✅ 基金规模、成立日期
- ✅ 费率信息（管理费、托管费）
- ✅ 资产配置（股票/债券/现金比例）
- ✅ 行业分布
- ✅ 交易限制（申购/赎回状态）

**使用场景**: 基金详情页、基金对比

---

### 2. fund-holding（持仓信息）

**功能**: 获取基金重仓股/债券持仓

**返回数据**:
- ✅ 报告日期
- ✅ 股票投资比例
- ✅ 前 10 大重仓股（代码、名称、持仓比例、环比变化）
- ✅ 前 10 大债券持仓
- ✅ 持仓变动（新增/增持/减持）

**使用场景**: 持仓分析、重仓股跟踪

---

### 3. fund-performance（历史业绩）

**功能**: 获取基金各阶段业绩表现

**返回数据**:
- ✅ 阶段收益率（近 1 周/1 月/3 月/半年/1 年/3 年/5 年/成立来）
- ✅ 今年来/近 1 年/近 3 年/近 5 年收益
- ✅ 四维指标分析（收益能力、抗波动、抗回撤、夏普比率）
- ✅ 同类排名百分比
- ✅ 年度收益

**使用场景**: 业绩走势、同类对比、四维雷达图

## 💡 技术亮点

### 1. 批量查询支持

所有接口支持单个或批量查询：
```typescript
// 单个查询
{ "codes": "000001" }

// 批量查询
{ "codes": ["000001", "000002", "000003"] }
```

### 2. 统一错误处理

所有接口统一返回格式：
```typescript
{
  success: boolean,
  data?: T,
  error?: string
}
```

### 3. 数据标准化

将盈米返回的复杂 JSON 结构转换为前端友好的格式：
- 字段名使用 snake_case
- 日期格式统一为 YYYY-MM-DD
- 百分比数据保留 2 位小数

### 4. 类型安全

使用 TypeScript 定义完整接口类型：
```typescript
interface FundDetail {
  fund_code: string;
  fund_name: string;
  fund_type?: string;
  // ...
}
```
