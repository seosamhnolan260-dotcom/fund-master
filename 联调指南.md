## 📡 接口列表

### 1. 基金搜索接口

**Endpoint**: `POST /functions/v1/fund-search`

**请求参数**：
```json
{
  "keyword": "华夏成长",  // 必填：基金代码/名称/拼音缩写
  "page": 0,              // 可选：页码，默认 0
  "size": 20              // 可选：每页数量，默认 20
}
```

**响应示例**：
```json
{
  "success": true,
  "data": [
    {
      "fund_code": "000001",
      "fund_name": "华夏成长",
      "fund_type": "混合型",
      "status": "正常申购",
      "setup_date": "2001-12-18"
    }
  ],
  "total": 1
}
```

**前端使用场景**：
- 搜索框自动补全
- 基金列表页

### 3. 基金详情接口

**Endpoint**: `POST /functions/v1/fund-detail`

**请求参数**：
```json
{
  "codes": "000001"        // 必填：单个代码或数组 ["000001", "000002"]
}
```

**响应示例**：
```json
{
  "success": true,
  "data": {
    "fund_code": "000001",
    "fund_name": "华夏成长",
    "fund_type": "混合型",
    "fund_invest_type": "灵活配置",
    "setup_date": "2001-12-18",
    "fund_company": "华夏基金",
    "fund_manager": "郑晓辉",
    "fund_scale": "29.37 亿",
    "risk_level": "中高风险",
    "min_invest_amount": 1,
    "management_fee": "1.5%",
    "custody_fee": "0.25%",
    "description": "...",
    "url": "https://..."
  }
}
```

**详细字段**（完整返回）：
- `summary`: 基金概要（净值、规模、成立日期等）
- `managers`: 基金经理列表（姓名、从业年限、管理收益）
- `assetPortfolios`: 资产配置（股票/债券/现金比例）
- `industryPortfolios`: 行业分布
- `tradeLimit`: 交易限制（申购/赎回状态、最低金额）

**前端使用场景**：
- 基金详情页基本信息
- 基金经理介绍
- 资产配置饼图

### 5. 历史业绩接口

**Endpoint**: `POST /functions/v1/fund-performance`

**请求参数**：
```json
{
  "codes": "000001"        // 必填：单个代码或数组
}
```

**响应示例**：
```json
{
  "success": true,
  "data": {
    "fund_code": "000001",
    "fund_name": "华夏成长",
    "yield_this_year": 0.28,     // 今年来收益率 %
    "yield_1y": 22.06,           // 近 1 年收益率 %
    "yield_3y": 15.87,           // 近 3 年收益率 %
    "yield_5y": -7.53,           // 近 5 年收益率 %
    "since_inception": 609.34,   // 成立来收益率 %
    "performance_periods": [
      {
        "period": "近 1 年",
        "growth_rate": 22.06,
        "rank": 50,              // 同类排名百分比
        "total_funds": 1000
      }
    ],
    "annual_returns": [
      {
        "period": "2025",
        "growth_rate": 15.30
      }
    ]
  }
}
```

**详细字段**（完整返回）：
- `metricsAnalyzes`: 四维指标分析（收益能力、抗波动能力、抗回撤能力、投资性价比）
- `stageReturns`: 各阶段收益（近 1 周/1 月/3 月/半年/1 年/3 年/5 年/成立来）

**前端使用场景**：
- 业绩走势图表
- 同类对比
- 四维雷达图

### 3. 前端集成

**Vue 3 示例**：
```vue
<script setup>
import { ref, onMounted } from 'vue'

const fundDetail = ref(null)
const loading = ref(false)

async function loadFundDetail(code) {
  loading.value = true
  try {
    const response = await fetch('/functions/v1/fund-detail', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ codes: code })
    })
    const result = await response.json()
    if (result.success) {
      fundDetail.value = result.data
    }
  } catch (error) {
    console.error('加载失败:', error)
  } finally {
    loading.value = false
  }
}

onMounted(() => {
  loadFundDetail('000001')
})
</script>
```

## ⚠️ 注意事项

### 1. 错误处理

所有接口统一返回格式：
```json
{
  "success": false,
  "error": "错误信息",
  "data": null
}
```

**常见错误**：
- `400`: 参数错误（基金代码格式不对）
- `404`: 未找到基金
- `500`: 服务器内部错误/MCP 调用失败

### 2. 批量查询

支持批量查询的接口：
- ✅ fund-detail
- ✅ fund-holding
- ✅ fund-performance

**示例**：
```json
{
  "codes": ["000001", "000002", "000003"]
}
```

### 3. 数据更新频率

- **净值数据**: 每个交易日 15:00 后更新
- **持仓信息**: 季度更新（财报披露）
- **基金详情**: 实时（但变动不频繁）
- **历史业绩**: 每个交易日更新
