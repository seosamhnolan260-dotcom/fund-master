### 3.1 fund-search 接口测试

**测试用例**: 精确搜索基金代码 000001

**请求**:
```json
{
  "method": "tools/call",
  "params": {
    "name": "SearchFunds",
    "arguments": {"keyword": "000001", "size": 5}
  }
}
```

```json
{
  "funds": [

      "fundCode": "000001",
      "fundName": "华夏成长",

      "returnYear": 0.1974200592,
      "url": "https://qieman.com/funds/000001"

}
```

**验证结果**:
- ✅ 响应时间: 251ms (< 500ms 要求)
- ✅ 数据格式: JSON 格式正确，包含 fundCode、fundName 等必需字段
- ✅ 数据准确性: 基金代码 000001 对应"华夏成长"，正确
