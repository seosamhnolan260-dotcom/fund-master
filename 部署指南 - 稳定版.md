## 📦 部署方式

### 方案 A: Cloudflare Pages（推荐）

#### 步骤 1: 初始化 Git 仓库

```bash
cd /Users/jun15prolan/.openclaw/workspace/projects/基金投资大师

git init
git add .
git commit -m "Phase 2 稳定版 - 先上线基础功能"
```

#### 步骤 2: 推送到 GitHub

```bash
# 创建新仓库（在 GitHub 上）
# 然后执行：
git remote add origin https://github.com/<your-username>/fund-investment-master.git
git branch -M main
git push -u origin main
```

#### 步骤 3: 配置 Cloudflare Pages

1. 访问 https://pages.cloudflare.com/
2. 点击 **"Create a project"**
3. 选择 **"Connect to Git"**
4. 选择 `fund-investment-master` 仓库
5. 配置构建设置：
   - **Framework preset**: `Create React App`
   - **Build command**: `pnpm build:h5`
   - **Build output directory**: `dist`
   - **Root directory**: (留空)
6. 点击 **"Save and Deploy"**

#### 步骤 4: 等待部署（5-10 分钟）

部署完成后获得地址：
```
https://fund-investment-master.<random>.pages.dev
```
